#### Tunnel
Ziel:

Ziel: 
* Poller entfernen

Ziel: Erhöhung der Verkehrssicherheit besonders für Fußgänger (Schulweg)
* Poller am Gehweg
Problem:
* Wegfall / Umwidmung Parkplätze Heiligenröder Str / Ecke Schuwirth
* Aufhebung Radwegebenutzungspflicht da Schwachsinn.
